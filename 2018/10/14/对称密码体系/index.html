对称密码体系

Posted on 2018-10-14

对称密码体系

主要特征：加密密钥和解密密钥相同
主要研究课题：密钥的产生/管理

流密码

流密码的安全强度完全取决于密钥的随机性和不可预测性
核心问题是：密钥流的产生
保持收发两端密钥流的精确同步是关键问题

基本思想：利用密钥k和密钥流发生器f生成一个密钥流
密钥流：Zi = f(k,δi)，δi是加密器中的记忆元件在时刻i的状态，f是以密钥k和δi作为输入参数的函数；

同步流密码
加密器中记忆元件的存储状态δi独立于明文字符
加密器：1、密钥流产生器 2、加密变换器

密钥流生成器的设计原则：

  • 足够长的周期
  • 高线性复杂度
  • 统计性能良好
  • 足够的“混乱”
  • 足够的“扩散”
  • 抵抗不同形式的攻击

在对称密码中，最重要的部分就是非线性部件

LFSR：线性反馈移位寄存器——流密码中产生密钥流的重要部分
级数stages：存储单元数n
状态state：n个存储单元的存数
反馈函数：f(state)
线性反馈移位寄存器：f是线性函数
非线性反馈移位寄存器：f是非线性函数

非线性序列
LFSR的输出不能直接作为函数，但是可以驱动一个非线性组合函数来产生密钥流：LFSR用了保证密钥的周期长度，平衡性等因素/非线性组合函数用来保证密钥流的各种密码性质，以抵抗各种可能的攻击

J-K触发器：是一个非线性的器件，输出不仅依赖于输入还依赖于前一个输出位

序列密码算法RC4

RC4的工作模式（OFB模式）

输出反馈模式（Output feedback, OFB）可以将块密码变成同步的流密码。它产生密钥流的块，然后将其与明文块进行异或，得到密文。与其它流密码一样，密文中一个位的翻转会使明文中同样位置的位也产生翻转。这种特性使得许多错误校正码，例如奇偶校验位，即使在加密前计算，而在加密后进行校验也可以得出正确结果。

随机密钥
1、密钥调度算法KeyScheduleAlgorithm
2、伪随机数生成算法PRGA
基本流程
1、密钥调度算法将一个随机密钥变换成一个初始置换
初始置换：

* 初始化S盒，线性填充S0=0, S1=1, …, S255=255
* S盒置换，交换Si和Sj，不断重复密钥K(L字节长) 
   j = (j + Si + K(i mod L)位)mod256.   j初始值为0

2、伪随机数生成算法利用初始置换生成一个伪随机数处序列

* 伪随机序列的生成：
* 伪代码：
    I := 0
    j := 0
    while GeneratingOutput:
    I := (I + 1) mod 256
    j := (j + S[I]) mod 256
    swap values of S[I] and S[j]
    K := S[(S[I] + S[j]) mod 256]
    output K
    endwhile

3、用伪随机数处理序列于plaintext进行XOR操作——加密/解密
